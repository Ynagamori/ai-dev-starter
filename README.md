# AI-Driven Dev Starter

AI を前提にした開発（AI 駆動開発）を行うための  
プロンプト・設定・テンプレート集です。

- ChatGPT / Claude / Gemini などのシステムプロンプト
- Codex / Cursor などに置く `AGENT.md` テンプレート
- AI を使った開発ワークフロー（設計 / 実装 / レビュー）
- アプリ種別ごとの AGENT テンプレ（Web アプリ / API サーバーなど）

「このリポジトリを開いて、そのまま AI に丸投げして開発を進める」
ことをゴールにしています。
