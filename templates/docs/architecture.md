# アーキテクチャ説明生成用プロンプト

以下の情報を AI に渡してください：

- ディレクトリ構成 (`tree` コマンドの出力など)
- 主要なエントリポイントファイルの内容
- 使用している主要ライブラリ / フレームワーク

## AI への依頼文テンプレ

> 上記の情報をもとに、このプロジェクトのアーキテクチャを
> 初見のエンジニアにも分かるように説明してください。
>
> - 構成図に相当するテキスト説明
> - 主要なレイヤー（フロント / API / DB など）の役割
> - データの流れ（ユーザー操作から DB まで）
> - 今後拡張しやすいポイント / 注意すべきポイント

このプロンプトを使って生成した説明文を、そのまま `docs/architecture.md` として保存できます。
