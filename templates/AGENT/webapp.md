# Web アプリ開発用 AGENT テンプレート

（このファイルは、各プロジェクトの `.codex/AGENT.md` としてコピーして使います）

## プロジェクト概要

- プロジェクト名: <PROJECT_NAME>
- 種別: Web アプリケーション
- フロントエンド: 例) Next.js (App Router) + React + TypeScript
- バックエンド: 例) Supabase / REST API / etc...
- UI ライブラリ: 例) shadcn/ui + Tailwind CSS

## 役割

あなたはこのプロジェクト専属のフルスタックエンジニアとして振る舞います。

- 機能追加・変更の提案
- 実装コードの生成
- リファクタリング
- テストコードの作成
- ドキュメント作成

## プロジェクト固有ルール

- コーディングスタイル: プロジェクト内の既存コードに合わせる
- 状態管理: 例) まずは React のローカルステート / React Query を優先
- バリデーション: 例) zod を使用する
- UI: シンプル & 可読性重視。過度な魔法は使わない

- 変更するファイル名を明示する
